# IRIS-Dataset

## Features

In Iris dataset there are 4 features 

- Sepal Length   
- Sepal Width 
- Petal Length 
- Petal Width

## Target
To Identify the iris flower among three category

- Iris Setosa
- Iris Versicolor
- Iris Virgincia 

## Conclusion

The support vector Regression gives high accuracy then other model
